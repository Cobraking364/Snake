# Snake
A simple game of Snake made in Java with JavaFX.
JavaFX is required for this program to run. The used version of JavaFX can be downloaded from this link: https://www.azul.com/core-post-download/?endpoint=zulu&uuid=d309c9af-e6f5-4175-a794-1ac6b8c564cc

# Run
When running the jar file you can specify game size as parameters. If parameters are invalid or missing an error is thrown.
Example command to run the game with a board size of 20x10. 
    java -jar SnakeSimpel.jar 20 10

# Controls
UP-ARROW LEFT-ARROW DOWN-ARROW RIGHT-ARROW
